四. 函数：

（1）函数的基本概念：

  1.函数就是为了方便我们维护代码以及固定代码块功能的重复使用，实际上就是一堆代码的集合，有了函数我们的开发效率也大大的提升，有太多太多好处了，我就不一一介绍了。
  2.函数有实参和形参的说法，并且只能实参向形参传递数据。
  3.形参概念：形参分为必选参数和默认参数，必选参数必须在默认参数的前面
  4.可变长参数：是任意多个参数，参数传递后会保存在一个元组中，可以通过for循环遍历处理,通过*参数名来定义。

（2）函数形参和实参：

函数的整体形式：              //其中形参列表和返回值不是必须出现。
def <函数名>(形参列表):      //必选参数和默认参数例子：  def zcl(name,age=17):     
    <函数语句块>               //可变长参数例子：def zcl( *tuple )
    return <返回值>

函数调用：                     
<函数名>（实参）              //实参传递为了代码可读性可以这样写：zcl(name=zhaoge,age=18)

三元表达式：
条件为真返回结果 if 条件判断 else 条件为假返回结果


lambda匿名函数：表达式                 //通常结合三元表达式使用
例子：fun = lambda x,y : x if x > y else y          
调用：fun(100,20)
其他玩法： fun = lambda : 函数()

作用域：
全局作用域：编写的整个脚本，例如一个变量的定义,可以在函数体内运行。
局部作用域：函数内的变量就是局部变量，只能作用于单个函数体内。 可以通过global将局部变量转为全局变量。

-------------------------------装饰器---------------------------------
产品经理需求：1. 超过20点不能玩游戏，如果没告诉时间则不能玩游戏。
装饰器的作用是不改变原有功能代码的基础上添加新的功能。
def time_control(fn):
    def Time_judgment(name,gaem,*args,**kwargs):
        clock = kwargs.get('clock',21)
        if clock <= 20:
            fn(name,gaem)
        else:
            print('晚上超过8点不能玩游戏哦！！！')

    return Time_judgment

@time_control
def play_gaem(name,gaem):
    print('{}玩的游戏是{}'.format(name,gaem))

play_gaem('赵创乐','王者荣耀')
play_gaem('李凤英','英雄联盟',clock=8)

（1）面向对象概述：

  面向对象的思想，实际上是将现实世界的事物，在电脑世界里进行细微的拆分，并最可能的贴合人类的思维方式，在python中创建一个类就等于创建一个模板，而我们可以通过这个模板可以创建不同的对象，但总的来说这些对象还是属于一大类型，并且不同的类所产生的对象实际上都是相关联的，面向对象的3大特性：封装性，继承性，多态性。

json序列化和反序列化：
python里数据存入文件只支持字符串和二进制。  主要作用：用于不同平台之间传输数据。
json可以将python里的数据（str/list/tuple/dict/int/float/bool/none）等转换为json数据，然后存入文件当中，读取文件时在将数据反序列化为python数据的某个类型。
import json                                    //导入json模块
json.dumps(name)                            //将数据转换成json字符串，不会保存到文件。
json.dump(name,file)                        //将数据转换为json字符串，并保存到指定文件。
json.loads(x)                                //将json字符串转换为python数据。
json.load(file)                                 //读取文件，将读取的内容转换为python数据

pickle序列化和反序列化：         主要作用：将数据原封不动的转换为二进制存储，只能在python中识别。
python里数据存入文件只支持字符串和二进制。
pickle可以将python里的数据（str/list/tuple/dict/int/float/bool/none）等转换为二进制数据。
import pickle                                //导入pickle模块

pickle.loads(x)                                //将二进制转换为python数据。
pickle.load(file)                            //读取文件，将读取的内容转换为python数据

string='WUdiDEge'

re.sub()                                    //显示替换后的结果,string类型。
re.subn()                                    //显示替换后的结果,tuple类型。

string='WUdiDEgeWU'

---------------------------------批量检测linux登录密码------------------------------
import paramiko
import os
import multiprocessing
import time
passwd=['123456',
        '123123',
        '666666',
        '4rsdsds',
        'sdsffss',
       ]
strart_time = time.time()
def ssh_client(host,Passwd):
    ssh = paramiko.SSHClient()
    ssh.set_missing_host_key_policy(paramiko.AutoAddPolicy())
    try:
        ssh.connect(hostname=host,port=22,username="root",password=Passwd,timeout=3)
        print("{}登录成功，密码是{}".format(host,Passwd))
    except Exception as e:
        print("Error：{} [{},尝试的密码是{}]".format(e,host,Passwd))
    ssh.close()

def main():
    with open('{}/ip.txt'.format(os.path.dirname(os.path.abspath(__file__)),'r',encoding="utf-8")) as F:
        ip_list = [x.strip('\n') for x in F.readlines()]
        po = multiprocessing.Pool(len(ip_list))
        for x in ip_list:
            for y in passwd:
                po.apply_async(ssh_client,(x,y))
        po.close()
        po.join()
        print ("TimeUsed:{}".format(time.time()-strart_time))

if __name__ == '__main__':
    main()
